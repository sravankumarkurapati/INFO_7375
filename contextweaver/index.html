üß† ContextWeaver

Advanced Multi-Document Reasoning Engine with Hybrid Intelligence

ContextWeaver is a next-generation generative AI system developed for INFO 7375 at Northeastern University. Unlike traditional Retrieval-Augmented Generation (RAG) systems that simply retrieve and concatenate documents, ContextWeaver performs true multi-document reasoning with uncertainty quantification, automated fact-checking, and intelligent hybrid retrieval.

üéØ Project Objective

Develop a sophisticated AI system that addresses the critical limitations of traditional RAG:

  • ‚ùå Problem: Traditional RAG cannot reason across multiple documents - it only retrieves and concatenates
  • ‚úÖ Solution: Multi-hop reasoning chains information through 2-3 iterative steps (85% confidence, 2.3√ó improvement over baseline)
  • ‚ùå Problem: Fails completely when queries are outside the knowledge base
  • ‚úÖ Solution: 3-tier hybrid retrieval (Local‚ÜíWeb‚ÜíLLM) ensures 100% query success with appropriate confidence levels (90%, 75%, 50%)
  • ‚ùå Problem: Cannot detect contradictions or provide confidence metrics
  • ‚úÖ Solution: Automated contradiction detection (95% confidence) + Bayesian uncertainty quantification with component breakdown
  • ‚ùå Problem: No fact-checking or verification of generated answers
  • ‚úÖ Solution: Automated fact-checking verifies all claims against sources (100% verification in testing)

üèóÔ∏è System Architecture

8-stage sequential pipeline with 3-tier hybrid retrieval

Pipeline Overview

ContextWeaver processes queries through 8 sequential stages, each validated in testing. The pipeline intelligently adapts processing based on source type (local, web, or LLM) and query complexity.

Stage Component Performance
1. Query Validation Format check, classification <0.01s
2. Hybrid Retrieval 3-tier fallback (Local/Web/LLM) <5s | 90%/75%/50% conf
3. Multi-Factor Ranking 5 factors weighted <1s | Score: 0.643
4. Graph Expansion NetworkX traversal <0.01s | +3 docs
5. Multi-Hop Reasoning Up to 3 hops 25.2s | 2 hops | 85% conf
6. Contradiction Detection Conflict identification 9.0s | 95% conf
7. Uncertainty Quantification Bayesian estimation <0.01s | 53.4% conf
8. Fact-Checking Claim verification 1.0s | 100% verified

Key Architectural Features

  • Modular Design: 11 independent modules with clear interfaces - each can function standalone
  • Loose Coupling: Components communicate through well-defined APIs - changes don't break system
  • Progressive Enhancement: Basic features work without all components - advanced features toggleable
  • Fail-Safe: 3-tier retrieval ensures no query fails - graceful degradation at each tier
  • Transparency: All decisions explained - reasoning chains visible, metrics exposed
